• Kdb4J

Kdb4j an event-based libary for kdb+ in Java providing a high-performance library for publishing and subcribing to kdb+ timeseries database.

  1. Non blocking IO using a single thread event loop
  2. Allocation free and Lock free

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git:github.com/isaiah-perumalla/kdb4j
